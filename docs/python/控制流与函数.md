# Python 控制流与函数

> Python 的控制流和 Java 逻辑一样，只是语法不同。最大的区别是用缩进代替大括号。

## 缩进规则

Python 用缩进表示代码块，不用大括号：

```python
# 正确
if True:
    print("缩进的代码属于 if 块")
    print("同一级缩进 = 同一个代码块")

# 错误（IndentationError）
if True:
print("没有缩进，会报错")
```

对比 Java：

```java
if (true) {
    System.out.println("大括号包裹的代码属于 if 块");
}
```

**缩进层级**通常用 4 个空格（或 1 个 Tab），但必须一致：

```python
# 正确
if True:
    print("第一层")
    if True:
        print("第二层")

# 错误（混用空格和 Tab）
if True:
    print("4个空格")
	print("1个Tab")   # 报错！
```

## 条件判断 if / elif / else

```python
age = 25

if age < 18:
    print("未成年")
elif age < 60:          # elif = Java 的 else if
    print("成年人")
else:
    print("老年人")
```

对比 Java：

```java
int age = 25;

if (age < 18) {
    System.out.println("未成年");
} else if (age < 60) {
    System.out.println("成年人");
} else {
    System.out.println("老年人");
}
```

### 条件表达式

```python
# Python 的条件表达式（三元运算符）
age = 25
status = "成年" if age >= 18 else "未成年"

# 等价于 Java 的
String status = age >= 18 ? "成年" : "未成年";
```

### 多条件判断

```python
# and = Java 的 &&
# or = Java 的 ||
# not = Java 的 !

age = 25
has_id = True

if age >= 18 and has_id:
    print("可以进入")

if age < 18 or not has_id:
    print("不能进入")
```

## for 循环

Python 的 for 循环更像 Java 的 for-each：

```python
# 遍历列表
fruits = ["苹果", "香蕉", "橙子"]
for fruit in fruits:
    print(fruit)

# 带索引遍历
for i, fruit in enumerate(fruits):
    print(f"{i}: {fruit}")

# 范围循环（类似 Java 的 for (int i = 0; i < 5; i++)）
for i in range(5):      # 0, 1, 2, 3, 4
    print(i)

for i in range(2, 5):   # 2, 3, 4
    print(i)

for i in range(0, 10, 2):  # 0, 2, 4, 6, 8（步长为2）
    print(i)
```

对比 Java：

```java
// Java 的 for-each
for (String fruit : fruits) {
    System.out.println(fruit);
}

// Java 的传统 for
for (int i = 0; i < 5; i++) {
    System.out.println(i);
}
```

### 遍历字典

```python
person = {"name": "张三", "age": 25, "city": "北京"}

# 遍历 key
for key in person:
    print(key)

# 遍历 value
for value in person.values():
    print(value)

# 遍历 key-value 对
for key, value in person.items():
    print(f"{key}: {value}")
```

## while 循环

```python
count = 0
while count < 5:
    print(count)
    count += 1       # Python 没有 ++，只能用 +=
```

### break 和 continue

```python
# break：跳出循环
for i in range(10):
    if i == 5:
        break        # 遇到 5 就停止
    print(i)         # 输出 0, 1, 2, 3, 4

# continue：跳过本次循环
for i in range(10):
    if i % 2 == 0:
        continue     # 跳过偶数
    print(i)         # 输出 1, 3, 5, 7, 9
```

和 Java 完全一样。

## 函数定义

```python
# 基本函数
def greet(name):
    return f"你好，{name}"

# 带默认参数
def greet(name, greeting="你好"):
    return f"{greeting}，{name}"

# 调用
greet("张三")              # "你好，张三"
greet("张三", "早上好")     # "早上好，张三"
```

对比 Java：

```java
// Java 不支持默认参数，需要重载
String greet(String name) {
    return "你好，" + name;
}

String greet(String name, String greeting) {
    return greeting + "，" + name;
}
```

### 返回多个值

```python
# Python 可以直接返回多个值（实际是返回 Tuple）
def get_location():
    return "北京", 39.9, 116.4

city, lat, lon = get_location()   # 解包
```

Java 需要创建一个类或使用数组。

### 可变参数

```python
# *args：接收任意数量的位置参数
def sum_all(*args):
    return sum(args)

sum_all(1, 2, 3)       # 6
sum_all(1, 2, 3, 4, 5)  # 15

# **kwargs：接收任意数量的关键字参数
def print_info(**kwargs):
    for key, value in kwargs.items():
        print(f"{key}: {value}")

print_info(name="张三", age=25, city="北京")
```

## 类定义

```python
class Dog:
    # 类变量（所有实例共享）
    species = "犬科"

    def __init__(self, name, age):     # 构造函数
        self.name = name               # 实例变量
        self.age = age

    def bark(self):                    # 方法
        return f"{self.name}在叫：汪汪！"

    def __str__(self):                 # 类似 Java 的 toString()
        return f"Dog({self.name}, {self.age}岁)"

# 创建实例
dog = Dog("旺财", 3)
print(dog.bark())        # "旺财在叫：汪汪！"
print(dog)               # "Dog(旺财, 3岁)"（调用 __str__）
```

对比 Java：

```java
class Dog {
    static String species = "犬科";   // 类变量
    String name;
    int age;

    Dog(String name, int age) {       // 构造函数
        this.name = name;
        this.age = age;
    }

    String bark() {                   // 方法
        return name + "在叫：汪汪！";
    }

    @Override
    public String toString() {
        return "Dog(" + name + ", " + age + "岁)";
    }
}
```

### self 的作用

`self` 类似 Java 的 `this`，但 Python 要显式写出来：

```python
class Dog:
    def __init__(self, name):   # self 是第一个参数
        self.name = name        # 通过 self 访问实例变量

    def bark(self):             # 方法也要写 self
        return f"{self.name}在叫"
```

Java 的 `this` 是隐式的，不需要写在参数里。

### 继承

```python
class Animal:
    def __init__(self, name):
        self.name = name

    def speak(self):
        return "..."

class Dog(Animal):              # 继承 Animal
    def speak(self):            # 重写方法
        return f"{self.name}: 汪汪！"

class Cat(Animal):
    def speak(self):
        return f"{self.name}: 喵喵！"

dog = Dog("旺财")
print(dog.speak())              # "旺财: 汪汪！"
```

## 异常处理

```python
try:
    result = 10 / 0
except ZeroDivisionError:           # 捕获特定异常
    print("不能除以零")
except Exception as e:              # 捕获所有异常
    print(f"出错了: {e}")
else:
    print("没有异常时执行")
finally:
    print("无论如何都执行")
```

对比 Java：

```java
try {
    int result = 10 / 0;
} catch (ArithmeticException e) {
    System.out.println("不能除以零");
} catch (Exception e) {
    System.out.println("出错了: " + e);
} finally {
    System.out.println("无论如何都执行");
}
```

主要区别：`catch` → `except`，`catch (Exception e)` → `except Exception as e`

## with 语句

自动管理资源，类似 Java 的 try-with-resources：

```python
# 自动关闭文件
with open("file.txt", "r") as f:
    content = f.read()
# 离开 with 块后，文件自动关闭

# 等价于
f = open("file.txt", "r")
try:
    content = f.read()
finally:
    f.close()
```

## 常见模式速查

```python
# 条件赋值
status = "成年" if age >= 18 else "未成年"

# 列表推导式
squares = [x**2 for x in range(10)]         # [0, 1, 4, 9, 16, 25, 36, 49, 64, 81]
evens = [x for x in range(10) if x % 2 == 0]  # [0, 2, 4, 6, 8]

# 字典推导式
square_dict = {x: x**2 for x in range(5)}   # {0: 0, 1: 1, 2: 4, 3: 9, 4: 16}

# 链式比较
if 18 <= age < 60:      # 等价于 age >= 18 and age < 60
    print("成年人")

# 解包
a, b, c = [1, 2, 3]
a, *rest = [1, 2, 3, 4]  # a=1, rest=[2, 3, 4]
```
