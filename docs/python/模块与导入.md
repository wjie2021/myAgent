# Python 模块与导入

> Python 用 `import` 导入其他文件的代码。和 Java 的 `import` 逻辑类似，但语法更灵活。

## 导入标准库

Python 自带很多标准库，直接 import 就能用：

```python
# 导入整个模块
import json
json.loads('{"name": "张三"}')

# 导入模块中的特定函数
from json import loads
loads('{"name": "张三"}')

# 起别名（简化调用）
import json as j
j.loads('{"name": "张三"}')
```

## 导入自己写的文件

### 同目录下的文件

```text
my_agent/
├── main.py
├── config.py
└── tools.py
```

```python
# 在 main.py 中导入 config.py 的变量
from config import API_KEY, BASE_URL, MODEL_ID

# 在 main.py 中导入 tools.py 的函数
from tools import get_weather, get_attraction
```

### 从子目录导入

```text
my_agent/
├── main.py
├── config.py
└── tools/
    ├── __init__.py    ← 必须有这个文件，才能当作包导入
    ├── weather.py
    └── search.py
```

```python
# 方式一：从包中导入模块
from tools import weather
weather.get_weather("北京")

# 方式二：从包中导入特定函数
from tools.weather import get_weather
get_weather("北京")

# 方式三：从模块中导入
from tools.weather import get_weather as gw
gw("北京")
```

### `__init__.py` 的作用

`__init__.py` 让 Python 把目录当作"包"来处理：

```python
# tools/__init__.py
# 可以是空文件，也可以在这里统一导入

from tools.weather import get_weather
from tools.search import get_attraction

# 这样在 main.py 中就可以直接：
from tools import get_weather, get_attraction
```

## 相对导入 vs 绝对导入

```python
# 绝对导入（从项目根目录开始）
from tools.weather import get_weather

# 相对导入（从当前文件开始）
from .weather import get_weather        # . 表示当前目录
from ..config import API_KEY            # .. 表示上一级目录
```

相对导入在包内部使用更方便，但只能在包内使用，不能在顶层脚本中使用。

## 文件操作

```python
# 读取文件
with open("data.txt", "r", encoding="utf-8") as f:
    content = f.read()           # 读取全部内容

# 逐行读取
with open("data.txt", "r", encoding="utf-8") as f:
    for line in f:
        print(line.strip())      # strip() 去除换行符

# 写入文件
with open("output.txt", "w", encoding="utf-8") as f:
    f.write("Hello, World!\n")

# 追加写入
with open("output.txt", "a", encoding="utf-8") as f:
    f.write("新的一行\n")
```

### 路径操作（推荐用 pathlib）

```python
from pathlib import Path

# 创建路径对象
p = Path("data") / "output.txt"   # data/output.txt

# 读取文件
content = p.read_text(encoding="utf-8")

# 写入文件
p.write_text("Hello!", encoding="utf-8")

# 判断是否存在
p.exists()          # True/False
p.is_file()         # 是否是文件
p.is_dir()          # 是否是目录

# 遍历目录
for f in Path(".").glob("*.py"):    # 当前目录所有 .py 文件
    print(f)
```

## 常见问题

### 模块找不到

```python
# ModuleNotFoundError: No module named 'xxx'

# 可能原因：
# 1. 模块名拼写错误
# 2. 没有安装（pip install xxx）
# 3. 文件不在 Python 搜索路径中

# 解决：检查文件位置和 import 路径
```

### 循环导入

```python
# a.py 导入 b.py，b.py 又导入 a.py → 循环导入错误

# 解决方案：
# 1. 重构代码，把共享的部分提取到第三个文件
# 2. 延迟导入（在函数内部 import）
```

### 导入顺序（PEP 8 推荐）

```python
# 1. 标准库
import os
import json

# 2. 第三方库
import anthropic
import requests

# 3. 本地模块
from config import API_KEY
from tools import get_weather
```

## Python vs Java 导入对比

| Python | Java | 说明 |
|--------|------|------|
| `import os` | `import java.io.*` | 导入模块 |
| `from os import path` | `import java.io.File` | 导入特定 |
| `import os as o` | 不支持 | 起别名 |
| `from . import x` | 同包内直接访问 | 相对导入 |
| `__init__.py` | 无（目录即包） | 包标识文件 |
