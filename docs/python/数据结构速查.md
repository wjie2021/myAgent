# Python 数据结构速查

Agent 开发中天天用的几种数据结构，遇到不懂的来查。

## List（列表）

```python
# 创建
fruits = ["苹果", "香蕉", "橙子"]

# 访问
fruits[0]        # "苹果"（第一个）
fruits[-1]       # "橙子"（最后一个）

# 添加
fruits.append("葡萄")    # 末尾追加

# 遍历
for fruit in fruits:
    print(fruit)

# 长度
len(fruits)      # 4
```

## Dict（字典）

Agent 开发中**最常用**，API 返回值基本都是 dict。

```python
# 创建
person = {
    "name": "张三",
    "age": 25,
    "city": "北京"
}

# 访问
person["name"]           # "张三"
person.get("name")       # "张三"（推荐，key 不存在不报错）
person.get("phone", "无")  # "无"（默认值）

# 修改
person["age"] = 26

# 遍历
for key, value in person.items():
    print(f"{key}: {value}")

# 判断 key 是否存在
"name" in person         # True
```

## 嵌套结构

API 返回值通常是 list 嵌套 dict：

```python
# 比如天气 API 返回
weather = {
    "city": "北京",
    "temperature": 31,
    "forecast": [
        {"date": "今天", "weather": "晴"},
        {"date": "明天", "weather": "多云"}
    ]
}

# 访问嵌套值
weather["city"]                  # "北京"
weather["forecast"][0]["weather"] # "晴"
weather["forecast"][0]["date"]   # "今天"
```

## JSON 与 Dict 的转换

API 返回的是 JSON 字符串，需要转成 dict 才能用：

```python
import json

# JSON 字符串 → Dict
json_str = '{"name": "张三", "age": 25}'
data = json.loads(json_str)     # {"name": "张三", "age": 25}
data["name"]                    # "张三"

# Dict → JSON 字符串
person = {"name": "张三", "age": 25}
json_str = json.dumps(person, ensure_ascii=False)  # '{"name": "张三", "age": 25}'
```

## 字符串格式化（f-string）

Agent 开发中拼接提示词常用：

```python
city = "北京"
temp = 31

# f-string 格式化（推荐）
msg = f"{city}今天{temp}度"       # "北京今天31度"

# 带表达式
msg = f"温度是{temp * 9 // 5 + 32}华氏度"
```

## 列表推导式

简化循环的写法，看到能读懂就行：

```python
# 传统写法
results = []
for item in data:
    results.append(item["name"])

# 列表推导式（一行搞定）
results = [item["name"] for item in data]

# 带条件筛选
hot_days = [item for item in data if item["temp"] > 30]
```

## 常见操作速查

```python
# 取第一个/最后一个
items[0]         # 第一个
items[-1]        # 最后一个

# 切片
items[1:3]       # 第2到第3个
items[:3]        # 前3个

# 判断是否为空
if items:        # 非空
if not items:    # 空列表

# 合并两个列表
a = [1, 2]
b = [3, 4]
c = a + b        # [1, 2, 3, 4]

# 去重
unique = list(set(items))
```

## Python vs Java 对比

| 操作 | Java | Python |
|------|------|--------|
| 创建列表 | `List.of(1, 2, 3)` | `[1, 2, 3]` |
| 创建字典 | `Map.of("k", "v")` | `{"k": "v"}` |
| 访问 | `map.get("key")` | `dict["key"]` 或 `dict.get("key")` |
| 遍历 | `for (var x : list)` | `for x in list:` |
| 长度 | `list.size()` | `len(list)` |
| 判空 | `list.isEmpty()` | `not list` |
