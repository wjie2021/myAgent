# Python 常见语法速查

看不懂 Python 语法时来查。对照 Java 理解更快。

## if / elif / else

```python
# Python 用缩进表示代码块，不用大括号
if age < 18:
    print("未成年")
elif age < 60:
    print("成年人")
else:
    print("老年人")
```

对比 Java：

```java
// Java 用大括号
if (age < 18) {
    System.out.println("未成年");
} else if (age < 60) {
    System.out.println("成年人");
} else {
    System.out.println("老年人");
}
```

## for 循环

```python
# 遍历列表
for item in items:
    print(item)

# 带索引
for i, item in enumerate(items):
    print(f"{i}: {item}")

# 范围循环
for i in range(5):      # 0, 1, 2, 3, 4
    print(i)
```

## while 循环

```python
count = 0
while count < 5:
    print(count)
    count += 1       # Python 没有 ++，只能用 +=
```

## 函数定义

```python
# 基本函数
def greet(name):
    return f"你好，{name}"

# 带默认参数
def greet(name, greeting="你好"):
    return f"{greeting}，{name}"

# 调用
greet("张三")              # "你好，张三"
greet("张三", "早上好")     # "早上好，张三"
```

对比 Java：

```java
// Java
String greet(String name) {
    return "你好，" + name;
}

String greet(String name, String greeting) {
    return greeting + "，" + name;
}
```

## 类定义

```python
class Dog:
    def __init__(self, name):     # 构造函数，self 类似 Java 的 this
        self.name = name

    def bark(self):               # 方法
        return f"{self.name}在叫"

# 使用
dog = Dog("旺财")
print(dog.bark())                 # "旺财在叫"
```

对比 Java：

```java
class Dog {
    String name;
    Dog(String name) { this.name = name; }
    String bark() { return name + "在叫"; }
}
```

## try / except（异常处理）

```python
try:
    result = 10 / 0
except ZeroDivisionError:
    print("不能除以零")
except Exception as e:
    print(f"出错了: {e}")
finally:
    print("无论如何都执行")
```

对比 Java：`try/catch/finally` 语法类似，只是 `catch` 变成了 `except`。

## with 语句（自动关闭资源）

```python
# 自动关闭文件，不需要手动 close
with open("file.txt", "r") as f:
    content = f.read()

# 等价于
f = open("file.txt", "r")
try:
    content = f.read()
finally:
    f.close()
```

类似 Java 的 try-with-resources：

```java
try (var f = new FileReader("file.txt")) {
    // 自动关闭
}
```

## import（导入模块）

```python
# 导入整个模块
import json
json.loads("{}")

# 导入特定函数
from json import loads
loads("{}")

# 起别名
import json as j
j.loads("{}")

# 导入自己写的文件
from config import API_KEY     # 从 config.py 导入 API_KEY
from tools import get_weather  # 从 tools/ 目录导入 get_weather
```

## None（空值）

Python 的 null 叫 `None`：

```python
result = None

if result is None:        # 判断是否为空（推荐用 is）
    print("没有结果")

if result is not None:    # 判断是否不为空
    print("有结果")
```

对比 Java：`None` ≈ `null`，`is None` ≈ `== null`

## 类型提示（可选）

Python 的类型提示不强制，但能让代码更清晰：

```python
# 不加类型提示
def greet(name):
    return f"你好，{name}"

# 加类型提示
def greet(name: str) -> str:
    return f"你好，{name}"

# 复杂类型
from typing import list, dict
def get_weather(city: str) -> dict:
    ...
```

| `and/or/not` | `&&/\|\|/!` | 逻辑运算 |
