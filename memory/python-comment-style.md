写 Python 代码时要多写注释，逐行或逐块解释逻辑。

**Why:** 用户是 Java 程序员，没有系统学过 Python，不熟悉 Python 语法和惯用写法。注释能帮助他理解代码含义。

**How to apply:**
- 变量声明、函数调用、循环、条件等都要加注释
- Python 特有的写法（如列表推导、with 语句、装饰器等）要特别解释
- 注释用中文，方便用户阅读
