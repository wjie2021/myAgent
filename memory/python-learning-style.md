用户是 Python 新手（Java 背景），但不想系统性学习 Python。

学习方式：
- 在学习 AI Agent 的过程中自然学习 Python
- 需要体系化的知识库，不仅仅是速查手册
- 每个 Python 概念都要有"为什么需要"的上下文，不能只是语法罗列
- 与 Java 对比能帮助理解

已创建的 Python 文档结构：
- 数据结构与类型.md — 系统学习
- 控制流与函数.md — 系统学习
- 模块与导入.md — 系统学习
- 依赖管理.md — 工具类
- 全局python与虚拟环境.md — 工具类
- python新手指南.md — 补充参考（含常见坑）

**Why:** 用户纯兴趣驱动，没有强制学习要求，希望学习过程平滑自然

**How to apply:** 讲解 Python 时结合 Agent 开发场景，说明"这个知识点在 Agent 开发中什么时候会用到"
